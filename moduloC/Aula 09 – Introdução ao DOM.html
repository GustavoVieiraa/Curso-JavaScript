JavaScript 09 – Introdução ao DOM

O que é DOM?

DOM (Document Object Model), está presente quando estamos utilizando o JavaScript dentro do nosso navegador.

Árvore DOM

Basicamente, todos os objetos HTML5 dentro do nosso navegador, em JavaScript é uma window.

    Dentro do nosso window, temos diversos outros objetos:
  • location
  • document
      • Dentro de document, temos um object, extremamente importante que é o nosso:
    • HTML, e nossos filhos HEAD e BODY
  • history

Podemos acessar os nossos elementos DOM, através de:

  • MARCA
  • ID
  • NOME
  • CLASSE
  • SELETOR
